## 一、项目概述

**Vibe TimeNews** 是一个每日新闻日报自动生成器，从多个权威信息源（VentureBeat AI、BBC、CNN、东方财富）自动采集最新资讯，生成一份精美的**深灰色极简风格** HTML 日报，帮助用户每天早上快速了解 AI 圈、金融圈、国际新闻三大领域的动态。

- **项目根目录**：`e:/Vibe-TimeNews/`
- **核心目标**：免费、稳定、易操作的本地日报生成工具
- **技术栈**：Python + httpx + feedparser + jinja2 + Windows 任务计划程序
- **发展阶段**：当前为 MVP 阶段，后续可扩展为 Web 应用和静态部署

## 六、核心特性（已实现）

- **热度排序**：内置关键词权重评分，突发/重磅/OpenAI 等关键词文章置顶
- **Tab 三栏切换**：金融圈 / AI 圈 / 国际新闻独立展示，点击切换
- **阅读全文精准跳转**：每条新闻链接原文页面，`target="_blank"`
- **深灰色极简风格**：Inter + Noto Sans SC 字体，纯列表布局 + SVG噪点纹理 + 波纹暗角
- **中文标题翻译**：英文 RSS 标题自动规则化翻译为中文
- **定时自动推送**：Windows 任务计划，每天 08:00 自动生成日报
