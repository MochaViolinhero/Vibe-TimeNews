## 一、项目概述

### 1.1 项目名称
**Vibe TimeNews** — 每日新闻实时推送日报生成器

### 1.2 项目定位
个人效率工具，从多个权威信息源自动采集、筛选、汇总每日最新资讯，生成一份精美的 HTML 日报，供快速浏览阅读。

### 1.3 核心价值
- **节省时间**：一个页面看完 AI、金融、国际三大领域动态
- **信息质量**：来源权威，过滤噪音，聚焦要点
- **定时自动**：每天早上自动生成，到公司即可阅读
- **离线可用**：HTML 文件本地保存，无网也能看

## 三、数据源方案

### 3.1 信息源列表

| 分类 | 来源 | 获取方式 | 地址 |
|------|------|----------|------|
| AI 新闻 | VentureBeat AI | 官方 RSS | `https://feeds.feedburner.com/venturebeat/AI` |
| 国际新闻 | BBC World News | 官方 RSS | `http://feeds.bbci.co.uk/news/world/rss.xml` |
| 国际新闻 | CNN World | 官方 RSS | `http://rss.cnn.com/rss/edition_world.rss` |
| 财经数据 | 东方财富（East Money） | 公开数据接口 | `https://push2.eastmoney.com` |

> **X（推特）暂不纳入**：官方 API 收费，第三方镜像不稳定，暂不加入 MVP。

### 3.2 数据采集策略
- **RSS 优先**：技术成熟、协议稳定、维护成本低
- **财经 API**：东方财富提供公开数据接口，若接口受限则降级为页面抓取
- **去重机制**：同一内容出现多次时去重，过滤纯广告内容
- **容错设计**：单一数据源失败不影响其他源采集，生成时标注数据来源

### 4.3 技术实现

- **HTML 模板引擎**：Jinja2（Python 内置，无需额外安装）
- **CSS**：内联到 HTML 文件，无需外部 CSS 文件，单文件即可完整展示
- **无 JavaScript 依赖**：纯 HTML + CSS，兼容性最强，打开即用
- **无需数据库**：所有数据在内存中处理，直接渲染输出

## 五、技术架构

### 5.1 系统流程

```
[定时触发 08:00]
       ↓
[数据采集层]  →  RSS 采集器（VentureBeat / BBC / CNN）
              →  财经采集器（东方财富）
       ↓
[数据整理层]  →  汇总 → 去重 → 分类 → 摘要截断
       ↓
[HTML 生成层] →  Jinja2 模板渲染
       ↓
[输出]        →  output/daily_YYYY-MM-DD.html
```

### 5.2 文件结构

```
e:/Vibe-TimeNews/
│
├── 📄 说明部分
│   ├── README.md              # 快速开始 & 使用说明
│   ├── PRD.md                 # 本文档，产品需求定义
│   └── STAGE_REPORT.md        # 阶段进展报告
│
├── 🗂️ src/                    # ═══ 代码执行层 ═══
│   ├── run.py                 # 主入口（定时 / 手动双击运行）
│   ├── fetcher/

│   │   ├── rss_fetcher.py     # RSS 采集（VentureBeat / BBC / CNN）
│   │   └── finance_fetcher.py # 东方财富数据采集
│   ├── aggregator.py          # 数据汇总、去重、分类
│   └── generator.py           # Jinja2 HTML 生成器（含热度排序）
│
├── 🗂️ scripts/                # ═══ 脚本层 ═══
│   └── run_daily.bat          # 定时任务执行脚本
```

## 六、运行与部署

### 6.1 手动运行（开发测试）

```bash
cd e:/Vibe-TimeNews
pip install -r requirements.txt
python src/run.py
# 生成 output/daily_2026-04-25.html
# 双击 HTML 文件用浏览器打开
```

### 6.2 定时自动运行

通过 Windows 任务计划程序，每天 08:00 自动执行：

1. 打开「任务计划程序」（Win + R → `taskschd.msc`）
2. 创建基本任务 → 命名为「Vibe-TimeNews 日报」
3. 触发器：每天 08:00
4. 操作：启动程序 → 选择 `pythonw.exe` 路径 + 参数 `e:\Vibe-TimeNews\src\run.py`
5. 完成配置

### 9.2 常见问题

**Q: 某天日报生成失败怎么办？**
A: 检查 `output/` 目录是否满或权限问题；手动运行 `python src/run.py` 重新生成。

**Q: RSS 源失效了怎么办？**
A: 在 `src/fetcher/rss_fetcher.py` 中替换为新的 RSS 地址，或在 STAGE_REPORT.md 中记录反馈。

**Q: 可以添加其他新闻源吗？**
A: 可以，在 `src/fetcher/rss_fetcher.py` 的 `RSS_SOURCES` 字典中添加新源即可。
